# __APP_NAME__

This starter is pre-wired for the Axionvera Vault contract on Stellar testnet.

## Run it

```bash
npm install
npm run dev
```

## Configure it

Copy `.env.example` to `.env.local` and update the contract ID if you want to point at a different deployment.

The default page uses:

- `axionvera-sdk`
- Tailwind CSS
- Freighter for browser wallet signing
