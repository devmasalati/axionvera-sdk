# create-axionvera-dapp

Scaffolds a Next.js starter app for Axionvera Vault workflows.

```bash
npx create-axionvera-dapp my-app
```

The generated project includes:

- Next.js with Tailwind CSS
- `axionvera-sdk`
- A Vault deposit/withdraw page wired for the Axionvera Testnet defaults
- A `.env.example` with the contract and RPC settings needed to get moving
